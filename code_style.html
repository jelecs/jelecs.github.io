Code Style

Naming

Variables

Variables use snake_case

Example:

int my_variable = 2;

Pointers

Place the * before the variable name

You can read it as the derefernced value of variable is type...

Example:

int *my_pointer;

Structs

Structs use PascalCase

Do not typedef structs unless special circumstances (currently none)

Example:

struct MyStructure { int my_member = 2; };

Functions

Functions use snake_case

If a function returns a pointer, leave a space

Examples:

int my_function() { return 2; }

Macros
