Thing

Make sure to include JEL

Always init before using any JEL thing and always quit before exiting the program

#include int main(void) { JEL_init(); JEL_quit(); return 0; }

Entities

There are some useful function and macros to know, but most of the stuff will be in components

JEL_Entity e = JEL_entity_create(); // Creates an entity JEL_entity_destroy(e); // Destroys an entity

Components

They're a little bit weird. You need to use JEL_COMPONENT_DEFINE and JEL_COMPONENT_CREATE.

Use JEL_COMPONENT_DEFINE in headers if you want to use components in multiple files

Always use JEL_COMPONENT_CREATE in a source file and only call it once per component

Use both of these macros outside of any functions/blocks

The example is for a single file program

#include JEL_COMPONENT_CREATE(Position, int, x, int, y) JEL_COMPONENT_DEFINE(Position, int, x, int, y) int main(void) { return 0; }

To attach components to entities use JEL_ENTITY_ADD

JEL_ENTITY_ADD(entity, Position);

To set values use JEL_ENTITY_SET

JEL_ENTITY_SET(entity, Position, x, 10);

To get values create a local variable and use JEL_ENTITY_GET

int my_x; JEL_ENTITY_GET(entity, Position, x, my_x);
